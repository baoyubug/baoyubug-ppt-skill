# 小红书清新风主题配色

本文件定义 PPT Skill 的默认视觉主题体系。采用小红书平台常用的清新、温暖、浅色调风格。

## 约束原则

- **只允许从以下预设主题中选择**，禁止自定义 hex 值
- 每个主题包含 6 个色彩变量，通过关键词触发
- 主题色通过 CSS 变量思想映射到 python-pptx 的 RGBColor

## 排版规范

| 元素 | 规范 |
|------|------|
| 标题字体 | 微软雅黑 Bold，28-44pt |
| 副标题字体 | 微软雅黑 Light，18-24pt |
| 正文字体 | 微软雅黑 Regular，14-18pt |
| 行间距 | 1.5倍 |
| 页面比例 | 16:9 |
| 留白原则 | 四周 ≥ 0.5 英寸 |

## 如何使用

在生成或编辑 PPT 时，引用主题名称：
- `"theme": "peach-pink"` — 蜜桃粉（默认）
- `"theme": "mint-green"` — 薄荷绿
- `"theme": "sky-blue"` — 天空蓝
- `"theme": "cream-yellow"` — 奶油黄
- `"theme": "lavender"` — 薰衣草紫

# 页面布局骨架

本文件定义 PPT Skill 支持的页面布局类型。每个布局包含结构描述和推荐使用场景。

---

## L01 · 封面 (Cover)

**用途**: 演示文稿第一页，传达主题和基调

```
┌─────────────────────────────────────┐
│                                     │
│           [装饰图形区域]              │
│                                     │
│        ┌──────────────────┐         │
│        │   主标题 (28-44pt)  │         │
│        │   副标题 (18-24pt)  │         │
│        └──────────────────┘         │
│                                     │
│          作者 / 日期                 │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "cover", "title": "...", "subtitle": "...", "author": "...", "date": "..."}`

---

## L02 · 目录 (Table of Contents)

**用途**: 展示整体结构，引导观众预期

```
┌─────────────────────────────────────┐
│  目录                                │
│                                     │
│  01  标题一  ──────────────────────  │
│  02  标题二  ──────────────────────  │
│  03  标题三  ──────────────────────  │
│  04  标题四  ──────────────────────  │
│                                     │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "toc", "items": [{"num": 1, "title": "..."}, ...]}`

---

## L03 · 标题 + 正文 (Title + Content)

**用途**: 最常用的内容页，适合大部分场景

```
┌─────────────────────────────────────┐
│  页面标题                            │
│                                     │
│  · 要点一                           │
│  · 要点二                           │
│  · 要点三                           │
│                                     │
│  补充说明文字...                      │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "title_content", "title": "...", "content": ["要点1", "要点2", ...]}`

---

## L04 · 左文右图 (Text Left + Image Right)

**用途**: 文字说明搭配配图，图文并茂

```
┌─────────────────────────────────────┐
│  页面标题                            │
│                                     │
│  ┌──────────┐   ┌────────────────┐  │
│  │          │   │                │  │
│  │  文字    │   │    图片        │  │
│  │  内容    │   │                │  │
│  │          │   │  16:9 或 4:3   │  │
│  │          │   │                │  │
│  └──────────┘   └────────────────┘  │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "text_image", "title": "...", "content": "...", "image_path": "..."}`

---

## L05 · 右文左图 (Image Left + Text Right)

**用途**: 图片在左，文字说明在右

镜像 L04，结构与 L04 一致，左右互换。

---

## L06 · 双栏对比 (Two Column Compare)

**用途**: A/B对比、前后对比、方案对比

```
┌─────────────────────────────────────┐
│  页面标题                            │
│                                     │
│  ┌──────────────┐ ┌──────────────┐  │
│  │   方案 A      │ │   方案 B      │  │
│  │              │ │              │  │
│  │  · 优点1     │ │  · 优点1     │  │
│  │  · 优点2     │ │  · 优点2     │  │
│  │  · 缺点1     │ │  · 缺点1     │  │
│  └──────────────┘ └──────────────┘  │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "two_content", "title": "...", "left_content": "...", "right_content": "..."}`

---

## L07 · 数据大字报 (KPI Big Number)

**用途**: 突出关键数字、里程碑数据

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│             85%                     │
│        用户满意度提升                 │
│                                     │
│      较去年同期增长 12 个百分点         │
│                                     │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "kpi", "number": "85%", "label": "用户满意度提升", "note": "..."}`

---

## L08 · 图片网格 (Image Grid)

**用途**: 展示多张图片，视觉丰富

```
┌─────────────────────────────────────┐
│  页面标题                            │
│                                     │
│  ┌─────┐ ┌─────┐ ┌─────┐          │
│  │     │ │     │ │     │          │
│  │  1  │ │  2  │ │  3  │          │
│  │     │ │     │ │     │          │
│  └─────┘ └─────┘ └─────┘          │
│  ┌─────┐ ┌─────┐ ┌─────┐          │
│  │  4  │ │  5  │ │  6  │          │
│  └─────┘ └─────┘ └─────┘          │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "image_grid", "title": "...", "images": ["path1", "path2", ...]}`

---

## L09 · 引用金句 (Quote)

**用途**: 突出重要观点、名言引用

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│    " 做难而正确的事，                  │
│     时间会给你答案。"                  │
│                                     │
│              —— 张一鸣               │
│                                     │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "quote", "text": "...", "author": "..."}`

---

## L10 · 结束页 (Ending)

**用途**: 演示结尾，致谢或行动号召

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│           感谢观看                    │
│                                     │
│       扫码关注 / 联系方式              │
│                                     │
└─────────────────────────────────────┘
```

**json字段**: `{"layout": "ending", "text": "感谢观看", "qrcode_path": "...", "contact": "..."}`

---

## 布局使用原则

- **封面 (L01)** 和 **结束页 (L10)** 是必须的，自动添加
- 正文避免连续 3 页相同布局，保护视觉节奏
- 超过 4 页时自动插入 **目录 (L02)**
- **数据大字报 (L07)** 和 **引用金句 (L09)** 作为节奏打破页，间隔使用
- **图片网格 (L08)** 中的图片建议统一长宽比

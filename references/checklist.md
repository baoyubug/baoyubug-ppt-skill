# 质量检查清单

PPT 产出前的自检项，按严重程度分级。

---

## P0 · 阻断级（必须全部通过才能交付）

- [ ] PPTX 文件可用 PowerPoint 正常打开，无损坏
- [ ] 所有文字可读（无乱码、无溢出、无重叠）
- [ ] 封面页和结束页都存在
- [ ] 标题不跨页截断
- [ ] 图片不越界遮挡文字
- [ ] 文件名符合用户预期命名

---

## P1 · 严重级（影响观感，必须修复）

- [ ] 字体统一（全篇使用不超过 2 种字体）
- [ ] 字号层级清晰（标题 > 正文 > 注释）
- [ ] 配色一致（全篇使用同一主题色系）
- [ ] 连续 3 页未出现相同布局
- [ ] 图片比例一致（同页多图时）
- [ ] 文字与背景对比度足够（浅底深字 / 深底浅字）
- [ ] 页面无大片空白（单页文字覆盖率 ≥ 30%）

---

## P2 · 一般级（理想情况下修复）

- [ ] 每页文字量适中（≤ 150 字，避免信息过载）
- [ ] 图片清晰度足够（≥ 800px 宽）
- [ ] 页码连续无缺失
- [ ] 目录页与实际页面对应
- [ ] 行间距适中（1.2x-1.8x）
- [ ] 列表项不超过 6 条
- [ ] 无语法错别字

---

## P3 · 锦上添花（有余力时优化）

- [ ] 封面有视觉冲击力
- [ ] 关键数据使用大字报突出
- [ ] 过渡页有章节分隔感
- [ ] 结束页有行动号召（二维码/联系方式）
- [ ] 装饰元素与主题色系和谐
- [ ] 动画/转场效果流畅（如有）

---

## 快速自检命令

生成完 PPT 后，对照以上清单逐项打勾。P0 和 P1 级别的项目全过才能交付给用户。

检查优先级：**P0 > P1 > P2 > P3**
